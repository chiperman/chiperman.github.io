Google Chrome 适合开启的 Feature flags

2019-11-23 次阅读 3 min read

桌面版 Chrome 值得体验的一些小 Features

开启方法

1.打开 Google chrome 浏览器
2.然后在地址栏框输入以下 URL

给标签页进行分组

开启方式：chrome://flags/#tab-groups

标签页过多，很难区分各个标签页的关联性，很容易就不小心关闭自以为无关的标签页，增加了反复打开一些标签页的问题。

如果使用了 给标签页进行分组 ，那么可以对同一类型标签页增加分组名并且用颜色区分。同时，在这个分组页面打开的新页面也属于这个标签页。当然也可以手动创建新的组或者是移除，只需要把标签页拖进 「组标记点」 的后面或者是移除 「组标记点」 就可以完成。

桌面端开启阅读模式

开启方式：chrome://flags/#enable-reader-mode

现在的页面信息量越来越多，失去了本质上的获取信息的这个理念，导致我们开发了越来越多的阅读模式扩展程序。

但是 Google Chrome 不像 Safari 自带阅读模式，Chrome 的阅读模式需要我们进入 Feature Flags 进行开启。

开启后在地址栏的右方有一个阅读模式按钮，点击后就可以打开阅读模式。



开启全局媒体播放控制

开启方式：chrome://flags/#global-media-controls

在浏览网页观看视频的时候，我们通常会打开多个多媒体页面。

但是 Chrome 并不支持你快速进行播放、暂停、快进或者是后退这些操作。

如果我们想进行这些操作，必须定位到当前页面才可以完成这些操作。

那么如果我们开启全局媒体播放控制，我们就可以在地址栏的右方对所有的多媒体页面进行快速以上操作。

鼠标悬停标签页展示网页预览图

开启方式：chrome://flags/#tab-hover-card-images

现在的人大多数都喜欢打开很多标签页进行浏览，那么一旦标签页过多，标签页的文本信息就会被挤压，你很难知道每个页面是什么，里面的大致内容是什么。

但是如果你开启了鼠标悬停标签页展示网页预览图，那么你可以不切换到这个标签页，只需要将鼠标悬停在标签页的上就可以快速预览这个标签页的具体内容。
