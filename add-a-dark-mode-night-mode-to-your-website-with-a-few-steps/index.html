🌓Add a Dark Mode / Night Mode to your website with a few steps.

2019-07-22 2 min read # Web

🌕 只需要几步就能为你的网页加上一个 🌓 暗黑模式/夜间模式 🌑

现在暗黑模式越来越流行，已经跳出程序员的圈子进入大众视野了。
不管是各种 IDE、手机 Apps、电脑软件和网页都越来越多的开始支持暗黑模式了，
如果想要快速的为你的网页加上一个轻量级的暗黑模式，
那么你就需要 Darkmode.js[1] 来帮助你了。
Darkmode.js 使用 css mix-blend-mode 将 Dark Mode 带入你的任何网站。
只需复制粘贴代码段，你就会获得一个 widget 来打开和关闭黑暗模式。

你可以通过点击 Darkmode.js 访问官网和查看文档以及 npm 的安装方式。

Step 1

复制这段代码

<script src="https://cdn.jsdelivr.net/npm/darkmode-js@1.4.0/lib/darkmode-js.min.js"></script>
<script>
  new Darkmode().showWidget();
</script>

把这段代码添加到你的 HTML 页面当中

Step 3

你已经成功为你的网站添加了暗黑模式
