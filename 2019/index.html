Review 2019

2019-12-31 次阅读 21 min read

🎉Review my 2019.

自从互联网公司发年度回顾后，
我也开始了一个小的年度系列，
今年的主题就叫做 「999」 吧。

我的 2019 相比 2018 其实感觉起来没有那么惨，
大多数都是好的回忆，当然也有不是那么顺利，
人生不就是这样么？

那么现在进入正题。

微博

今年特别关注了两个博主

@CBVIVI

一个是我最爱的 Vlog 博主，带我爱上了日本 🇯🇵 还有「平凡料理」也是跟他学的，爱上这些简单的料理，成为编程外的第二个爱好。他就是那种他到哪里，哪里就是日本 🇯🇵 的感觉

好名字真的可以让别人更容易记住你

是真的。

@FLYPIG

另一个则是让我爱上了胶片相机 📷， NOMO 绝对是我最爱的拍照软件。

他其实在网络上一直是一个恶霸的人设，其实每两周都等着他发布新的 NOMO 相机。

他今年发布的相机中，我最爱的 📷 是 #FR2 ，年底的快乐没有迟到。

编程 💻

Github

大一的时候我就已经开始用 Github ，只是那个时候没有太多机会，也不会操作，只是简简单单的注册了一个账号，没有干其他的事情。

可是今年开始，为了使用 Github Pages 配合 Gridea 搭建博客，我才开始学着使用 Git 以及 Github 进行代码托管。

然后就是下半年的 Android 开发课的项目，我就开始高强度使用 Github 提交代码、给别人的 repo 提交 PR、查看别人已经造好的轮子然后加入自己的项目等等。算是一个小成就吧，但是还是希望能写出高质量代码，不然也没啥意义。

Android 项目

花了一点心思做了一个 Android App，用来进行订阅管理，其实参考了 OneBox 和 Bobby 这两个软件的界面，但是还是花了很多心思，毕竟第一次做一个自己的 App 是我的小梦想吧。如果有机会我还想慢慢完善它。也不需要多少人用，能解决一些小痛点就好了。

博客

今年 5 月 7 号开始用 Gridea 写博客。

博客其实是一个挺老的东西，现在都被朋友圈和微博替代了。
那为什么还要写一个博客呢？
有些时候你得下定决心做一些事情，一开始可能很难，后面就习惯了。就算没人看，用来记录生活、整理学习笔记都是非常好的。
本身我就不是一个爱发朋友圈这些的人，那有些东西就可以写在这里。
今年最喜欢的文章一定是 「平凡料理」。

阿里云

今年 6 月左右白嫖了阿里云的一个 .design 域名以及一个轻量云服务器。

一开始我没想好干嘛，只是给我的博客用了这个域名，后来用 WebStack 建立了一个自己的 网址导航 用来整理自己的书签以及一些网址，一开始在服务器上搭建遇到了问题，还是找了很多解决办法，最后找了一个相对简单的办法成功搭建了这个网址导航，就算后面到期了，还是可以直接转为静态页面。

到了年底的时候，我们的笔记项目小组需要买域名，就在我的阿里云帐号里买了一个 iwriternote.com 的域名。明年就能上线了，Hope so.

软件

NOMO

这可以算是我这两年来最喜欢的 App 没有之一。拟物、胶片还有相机 📷 设计风格全都符合我的审美还有我喜欢的用户体验。

今年喜欢的还是年底出的 #FR2 和 ROMA 还有 CAM BOY 这三款相机，结果三款都是联名款。

年底还收获了 #FR2 这款潮流相机。可惜我比较喜欢的 MOON 是 PRO 用户才能使用的，如果单卖我可能就直接购买了。

其中 ROMA 可是我大网飞的今年的奥斯卡年度佳作，可惜我一直没找到好适合的时间去看，等等 2020 吧。

#FR2

CAM BOY

今年年初和大学同学去上海的玻璃博物馆参观，用 CAM BOY 拍了一张我很喜欢的照片，然后长时间用作各大社交软件的头像，毕竟是我最爱的像素风。

年底 少数派 上线了一个网页 NOMO 相机陈列馆 是和 NOMO 和合作的一个网页，可以看到里面 @flypig 讲诉的相机故事。

Telegram

2019 年一度逃离微信，使用 Telegram，这个我认为目前为止，最棒的即时通讯软件。

可惜因为国内想要使用要有网络条件支持， 所以并不能使用，至少对大多数人来说是不可用的。

再加上国内的社交软件已经被腾讯系占领了，无论是 QQ 还是微信。所以我还是不能彻底避免这个都 2019 了，还是如此难用的微信。即使微信的用户体验如此糟糕可是仍然有大批人被它绑架，或是被那些被它绑架而绑架的人，还有甚至是为了体现自己的成熟而放弃在多个方面碾压它的 QQ 本身就是一个难以理解的事情。张小龙就别封神了，他到底用不用自家的微信我都保持怀疑，毕竟这个软件太差劲了。

你现在甚至可以说「我教张小龙做产品！」

没想到老罗离开了锤子科技， 还挺伤感。毕竟以后听不到手机节的相声了。

2019 年度 游戏 「塞尔达：荒野之息」-「Florence」- 「勇敢的心」- 「马里奥：奥德赛」……

Nintendo Switch

2019 年买了人生中第一台游戏机 🎮 Nintendo Switch 成为了一个任吹。

然后就是一发不可收拾，除开一下图片，还有比如「马里奥：奥德赛」-「马里奥派对」都是已经回血❤️了的。

塞尔达传说：旷野之息

玩到了这辈子最好玩的游戏「塞尔达传说：旷野之息」。可能很多年都玩不到这么令人惊艳的游戏了吧。
还好我没有错过它太久。

我以前一直不怎么喜欢单机游戏，毕竟一个人玩太无聊了，加上大部分单机都是闯关模式。

但是!

但是不适用于 BOW，我第一次打开游戏，走出山洞的时候。伴随着 BGM 以及一个广角镜头，一瞬间就让我知道，就是它了，我会爱上它的。

买了 Switch 的那一个月，我整个人都魔怔了。每天下午躲在卧室，连上电视机 📺，拉上窗帘关掉手机，整个人都沉浸在海拉鲁大路上奔驰翱翔。

那一个月，我的收集癖，苹果、鱼还有还有各种东西都让我控制不住自己。

救公主？不存在的，至少在我收集完所有的东西之前是不可能的，毕竟我可是海拉鲁恶霸。

手游系列

我其实不是一个太爱玩手游的人，或者说不是一个爱玩手机网络游戏的人。

我玩得手游大多数是很古怪或者是很小众化的游戏。

我特别喜欢玩独立游戏。

一个制作者或者一个小团队，往往能够制作出超乎你预先的游戏。

Florence

Florence 就是这么一个游戏，它甚至不算一个游戏，只是一个可互动的故事。

但是就是这么一个游戏，反而让你欲罢不能。
「放手」这一章是我停留时间最长的，每次都不舍得让女主越走越快，知道男主消失。即便不放手就不能继续，但是我还是不愿意就这么结束。

真的值得花上半个小时玩一玩。

孩子们

孩子们 和 Florence 几乎一样，算不上游戏，也可以当做是漫画，可互动式漫画。

也可以独自一人，

美剧—Netflix

今年迷上了 Netflix - 「怪奇物语」还有互动版「黑镜：潘达斯奈基」简直太棒了。

年初的时候，「黑镜」电影版在 Netflix 上映，一开始我的代理看不了，然后花了很多力气到处找办法，终于体会到了互动版的精彩，五个结局，无数个分支故事路线配合网飞流畅的播放整个体验都太棒了。

后来三月份找到一个美国原生 IP 的服务商，从此沉迷 Netflix。

今年在网飞看了很多好看的剧「怪奇物语」-「纸钞屋」-「Rick and Morty」-「爱，死亡和机器人」-「Sex Education」还有一些就不值得提了。

「怪奇物语」

可能最好看的还得是 「怪奇物语」 。我都没法形容，各种彩蛋 🥚，80s 年代的风格本来不可能是我的风格，加上剧名有点点恐怖的感觉，让我一直拒绝看他。可是后来吧网飞天天推荐，那就看看呗，然后就真香。

每集都让我放不下，加上 小 11 和 Mike 的甜甜的故事，还有 Dustin 漏风的嘴巴以及全场吐槽的 Lucas，当然还有最倒霉的 Will 每个人我都好喜欢。我等到今年第三季的时候，因为网飞都是一次性放出，我专门提前下载好，星期六买了个麦当劳套餐然后回家一个人，真的是关掉手机然后关上门窗，整整看了 8 个多小时。从下午看到晚上再回学校。

除此之外，还有许多已经结束的美剧，其实还挺难过，毕竟都是一集一集看过来的，许多记忆就到此结束了。
「去他*的世界」- 「硅谷」- 「我是僵尸」- END

「去他*的世界」

James 和 Alyssa 的关系就是那种羡慕但是你不可能做到的那种，从一开始 James 想要杀掉 Alyssa，而 Alyssa 则是一个叛逆的女生。

从第一季结尾，James 就表露出他真的爱上了 Alyssa，然后在第二季其实他有一点点现在我们说的舔狗的意思。可是，结局总是美好的。

「硅谷」

我也是打算一天看完的，不准备一周一集，所以期末有一次考试第二天我就一次性刷完了。剧终季感觉结尾有点太快了，可能是我太喜欢它了，所以不舍得。最后一集大家回到 当时的小屋里面，围着桌子一点点回忆，不过也的确是很棒的结局，没有那么的成功，但是大家还是在一起。

结局也是回到桌边，玩着这个剧一开始的游戏，从开始到结束。

Always Blue！

我其实是大二才看的，那一个月刷完了前面的五季，然后到现在，虽然时间很短，但是是我很喜欢很喜欢的一部美剧。很幸运我也是学计算机的，一开始大家都是抱着 Change the world 的想法，希望不会被现实打败吧。希望而已。

明年还有带我进入美剧这个坑的 「神盾局特工」也要结束了，这真的是我最难过的，希望明年 Fitz 和 Simmons 不会再分离了吧。这一部分留到明年的总结再说吧。

也算是弥补当年没有看成的「你的名字。」吧，毕竟是带我爱上日本的动画。可能也是受了 Cbvivi 的影响吧？

年底平安夜我们俩又一起去看了「误杀」，看的时候就很熟悉，虽然一度猜到剧情走向，但是还是很好看，我觉得是我今年看到的国产电影第一梯队中的电影了。可惜结局是一个正能量向的结局，其实我更倾向暗黑一点的结局，不过可能过不了审，但是也挺不错了。
