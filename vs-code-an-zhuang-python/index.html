VS CODE 安装 python

2019-08-01 次阅读 1 min read

软件安装

安装 Python

首先，我们安装 Python 到自己的电脑里。
切记，一定要记得勾选添加环境变量。
不然还得手动去配置 Python 的环境变量。

检查 Python 是否安装成功

检查方法如下：
1、打开 cmd，输入 python，点击回车。
2、输入 import this
3、退出、输入 pip list，是否显示第三方的包。

安装第三方的包

通过 Python 自带的 pip 工具，安装配置 VScode 需要的包。
flake8 和 yapf
flake8 会检查编写代码时的不规范的地方和语法错误。
yapf 是一个代码格式化工具，可以一键美化代码。
安装过程如下：
1、在 cmd 输入 pip list 检查是否已存在需要的包
2、输入 pip install flake8 下载安装 flake8
3、输入 pip install yapf 下载安装 yapf

运行

打开 VScode 选择打开或选择目录
新建一个为 demo.py
输入一句

print("Hello VScode")

连续按两次F5执行文件。
输出
Hello VScode
