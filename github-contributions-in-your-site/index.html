如何给你的网站加上 Github Contributions Chart

2019-09-01 次阅读 1 min read

为你的网页加上你的 Github 贡献表

Github 贡献图标算得上 程序员 日历了，但是仅仅局限于 Github 并不能满足我们
如果想要在自己的网页或者博客加载这个 可视化数据表

那么你就需要这个 githubchart-api
你可以查看官网
也可以查看Github 仓库地址
获得相关信息

这个是我的提交记录，要知道这个并不是一张图片，而是实时更新的

使用方法

在你的网页中插入这段代码
<img src="https://ghchart.rshah.org/github-username" />
将 github-username 替换为自己的 Github 用户名即可

同时也支持修改配色，不再是默认的绿色方格。

自定义颜色

例如，如果你想要一个基于十六进制颜色的粉色主题图表 #EA4C89
你只需要在标签中加入颜色的十六进制
<img src="https://ghchart.rshah.org/EA4C89/github-username" />
